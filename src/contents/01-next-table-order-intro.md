## 4. 기존 테이블 오더의 문제점 분석 & 개선 방향

> ❌ **기존 테이블 오더 서비스의 문제점**

1️⃣ **언어 전환 속도가 느리다.** → 서버 요청을 줄여야 함  
2️⃣ **이미지 로딩이 너무 느리다.** → Cloudflare R2로 최적화  
3️⃣ **사용자 행동 데이터가 부족하다.** → 클릭 트래킹, 주문 패턴 분석 추가

✅ **이걸 전부 해결하는 제품을 만들자.**
